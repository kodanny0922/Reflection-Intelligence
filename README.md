# Reflection-Intelligence
A metacognitive framework for post-inference reasoning, enabling AI systems to reflect, analyze, and improve their own reasoning through a structured reflective loop. 

一個面向推理後階段的元認知框架，通過結構化的反思迴圈，使 AI 具備自我檢視、自我分析與自我改進的能力。
